SECCIÓN I - Preguntas Teóricas

1. ¿Cuál es la diferencia entre print y return en una función?
    Print muestra en pantalla pero no guarda el valor; return devuelve el valor para guardarlo o reutilizarlo.

2. ¿Qué imprime el siguiente código?
    Imprime el valor de 50.

3. ¿Qué sucede al ejecutar el siguiente código?
    Error

4. ¿Cuál es la forma correcta de importar SOLO la función sqrt de la librería math?
    from math import sqrt

5. ¿Por qué NO se recomienda usar from math import *?
    Porque importa todas las funciones y puede causar conflictos de nombres con tu código.

6. ¿Cuál es la diferencia entre una librería estándar y una librería externa?
    Las estándar ya vienen con Python, las externas se instalan con pip install.
